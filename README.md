# Binance Spot Testnet Trading Bot

A CLI-based automated trading bot that supports:

- ✅ Market Orders
- ✅ Limit Orders
- ✅ Stop-Limit Orders
- ✅ TWAP (Time-Weighted Average Price)
- ✅ Grid Strategy (Buy low / Sell high)
- ✅ Structured Logging
- ✅ Clean CLI interface
